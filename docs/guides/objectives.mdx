Use `pursue()` for work that must survive process restarts, wait without spending tokens, react to external events, or require durable plans and approvals.

## Create and pursue

```python
from aeon import Agent, Budget, Objective, Playbook, PlaybookStep, Policy, Runtime

agent = Agent(
    name="release-operator",
    models=["claude-sonnet-4-6"],
    policy=Policy(
        approval_required_tools=["deploy"],
        tool_risk_levels={"deploy": "critical"},
        workspace_only=True,
    ),
)

objective = Objective(
    goal="Keep this project releasable and publish verified releases.",
    context="React to repository.changed events.",
    success_criteria=["Tests pass", "Release evidence is recorded"],
    budget=Budget(max_cost_usd=20, max_cycles=10_000, max_turns_per_cycle=20),
    playbook=Playbook(
        name="verified-release",
        steps=[
            PlaybookStep("Inspect changes"),
            PlaybookStep("Run verification"),
            PlaybookStep("Publish with approval"),
        ],
    ),
)

run = agent.pursue(objective)
# equivalent: Runtime().pursue(agent, objective)
```

## Events and control

```python
run.emit(
    "repository.changed",
    {"ref": "main"},
    dedupe_key="push-123",
)

run.pause("Maintenance window")
run.resume()
run.cancel("Objective retired")

snapshot = run.snapshot()
idle = run.wait_until_idle(timeout=120)
terminal = run.wait(timeout=3600)
```

Use `dedupe_key` so retries do not create duplicate wake signals.

## Durable ledgers

```python
run.plan()       # work graph
run.events()     # wake / event journal
run.memories()   # working, episodic, semantic, procedural
run.actions()    # policy and idempotency-aware tool ledger
run.outcomes()   # measured results and evidence
```

## Approvals

When a tool requires approval, the objective waits for a durable human decision:

```python
for approval in run.approvals(pending_only=True):
    run.approve(approval["id"], note="Reviewed by release owner")
    # or run.reject(approval["id"], note="Unsafe during freeze")
```

## Reconnect after restart

```python
from aeon import Runtime

runtime = Runtime()
run = runtime.get_objective("objective-id")
waiting = runtime.list_objectives(statuses=["waiting", "blocked"])
```

`Runtime.list_objectives()` is independent from legacy one-off `Runtime.list()` runs.

## Budgets

| Field | Meaning |
| --- | --- |
| `max_cost_usd` | Total model spend across all cycles |
| `max_cycles` | Maximum wake cycles |
| `max_turns_per_cycle` | Hard turn limit per wake |
| `max_minutes` | Active execution minutes (dormant time excluded) |
| `max_tool_calls` | Total tool invocations |

See [policy and approvals](/guides/policy-and-approvals) for risk gates and containment.
